Overview
--------

This project pads nucleotide sequences.

Links
-----

- Download: https://pypi.org/project/seqpad/#files
- Index: https://pypi.org/project/seqpad/
- Source: https://github.com/johannes-programming/seqpad/
- Website: https://seqpad.johannes-programming.online/

Installation
------------

To install seqpad, you can use pip. Open your terminal and run:

.. code-block:: shell
    
    pip install seqpad

Features
--------

``seqpad.main``
~~~~~~~~~~~~~~~

The project also provides a CLI that can also be accessed from within Python. For more information, read the help message that can be accessed through one of the two following ways.

.. code-block:: shell

    # bash
    python3 -m seqpad -h

.. code-block:: python

    # python
    import seqpad
    seqpad.main(['-h'])

``seqpad.seqpad(seq: Any) -> Any``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The core function of this project is implemented as follows:

.. code-block:: python

    def seqpad(seq: Any) -> Any:
        while len(seq) % 3 != 0:
            seq += "N"
        return seq

``seqpad.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its test function.

.. code-block:: python
        
    import seqpad
    seqpad.test()
